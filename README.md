# 🍬 Candy Crush Game

A fun and interactive **Candy Crush Clone** built using **React + TypeScript**.  
Match candies, score points, unlock levels, and enjoy smooth animations with a modern UI.

---

## 🚀 Features

- 🎮 Match-3 gameplay mechanics
- ✨ Smooth animations & effects
- 📈 Score tracking system
- 🎯 Multiple levels with goals
- 💡 Hint system for possible moves
- 🔄 Restart & replay support
- 🎨 Modern colorful UI
- ⚡ Built with React Hooks
- 📱 Responsive design

---

## 🛠️ Tech Stack

- React
- TypeScript
- TSX
- CSS / Inline Styling

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/candy-crush-game.git
```

Go to the project folder:

```bash
cd candy-crush-game
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```

---

## 🎮 How to Play

1. Click on a candy.
2. Click an adjacent candy to swap.
3. Match 3 or more candies.
4. Earn points and complete level goals.
5. Finish all levels to win the game 🏆

---

## ✨ Game Features Explained

### 🎯 Levels
Each level has:
- Target Score
- Limited Moves

### 💡 Hint System
If the player is idle for a few seconds, the game highlights a possible move.

### 🔥 Combo System
Chain matches increase combo multiplier and score.
